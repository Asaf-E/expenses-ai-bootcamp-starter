# Bootcamp Prerequisites and Tools

## Before You Start

This tutorial is built around **AI-assisted development with Spec Kit**.

To complete the exercise, you should have the following ready before the session starts.

---

## Required Tools

### 1. Development Environment (IDE)
You need an IDE where you can:
- open the starter project
- edit Markdown and code files
- work with your AI assistant

Examples:
- https://code.visualstudio.com/
- Any IDE with AI assistant support

---

### 2. GitHub Copilot
GitHub Copilot is required for this tutorial.

You should:
- be signed in
- have Copilot enabled in your IDE

More info:
- https://github.com/features/copilot

---

### 3. Spec Kit
Spec Kit is the tool used to support **spec-driven development**.

It helps you:
- describe features
- create structured designs
- guide AI-assisted implementation

Official resources:
- Installation guide: https://github.github.com/spec-kit/installation.html
- Quickstart: https://github.github.com/spec-kit/quickstart.html
- Quick command to install and apply on current project: uvx --from git+https://github.com/github/spec-kit.git specify init --here

---

## Additional Requirements

- Python (required by Spec Kit setup)
- npm
- Git (recommended for project setup)
- Access to the starter project
- Ability to run the project locally
- Hint: Use the AI to install missing project dependencies after you have the IDE + Copilot :)

---

*Goal: arrive with the tools ready so the session can focus on design-first AI usage rather than environment setup.*
